Se instaló exitosamente el servirod web Apache en Pardus. Ahora es posible agregar páginas HTML locales en este directorio.

Pardus es una distribución Linux, libre para usar, modificar y distribuir, en la cual se puede ejecutar el servidor web Apache de manera segura. Puede usar el logo de Apache de abajo en sus páginas web mientras está usando el servidor Apache. 

Gracias por usar Pardus y Apache!
