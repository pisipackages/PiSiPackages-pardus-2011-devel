Você instalou com sucesso o servidor web Apache no Pardus. Agora é possível adicionar páginas HTML localizadas neste diretório.

Pardus é uma distribuição Linux livre para ser usada, modificada e distribuída na qual você pode executar o servidor web Apache de uma forma segura. Você pode usar o seguinte logo Apache enquanto o utilizar como servidor nas suas páginas web. Obrigado por utilizar o Pardus e o Apache!
