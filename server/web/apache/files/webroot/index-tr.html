Bu sayfayı gördüğünüze göre Pardus üzerine Apache Web Sunucusu'nu başarılı bir şekilde kurdunuz demektir. Bu sayfanın bulunduğu dizine kendi sayfalarınızı ekleyip hemen yayımlamaya başlayabilirsiniz.

Pardus, Apache Web Sunucusu'nun güvenli bir şekilde çalışabileceği özgür bir Linux dağıtımıdır.

Apache Web Sunucusu'nu kullanırken aşağıdaki Apache logosunu kullanabilirsiniz. Pardus ve Apache'yi kullandığınız için teşekkür ederiz!
