PARDUS Web Sunucusu Deneme Sayfası

Bu sayfanın hazırlandığı diğer diller: [İngilizce]

Bu sayfayı gördüğünüze göre Pardus üzerine Nginx Web Sunucusu'nu başarılı bir şekilde kurdunuz demektir. Bu sayfanın bulunduğu dizine kendi sayfalarınızı ekleyip hemen yayımlamaya başlayabilirsiniz.

Pardus, Nginx Web Sunucusu'nun güvenli bir şekilde çalışabileceği özgür bir Linux dağıtımıdır. Nginx Web Sunucusu'nu kullanırken aşağıdaki Nginx logosunu kullanabilirsiniz. Pardus ve Nginx'i kullandığınız için teşekkür ederiz!
